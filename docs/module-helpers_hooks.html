(static) getStatisticsHooks(collection) → {Object}

Source:
  • helpers/hooks.js, line 189

生成mongoose的统计函数hooks，包括：save, findOne, findOneAndUpdate, findOneAndRemove, count, find， 在初始化model时，添加hooks，便于监控mongodb的各类操作性能

Example
cosnt schema = xxx; // mongoose schema;
const hooks = require('./helpers/hooks');
const statisticsHooks = hooks.getStatisticsHooks('user');
_.forEach(statisticsHooks, (hooksInfos, hookName) => {
  const {
    pre,
    post,
  } = hooksInfos;
  if (_.isFunction(pre)) {
    // pre hook
    schema.pre(hookName, pre);
  }
  if (_.isFunction(post)) {
    // post hook
    schema.post(hookName, post);
  }
});

返回包括各个函数的hook，{save: function, findOne: function, ...}

(inner) fillData(opts, data) → {Object}

Source:
  • helpers/hooks.js, line 106

自动填充数据，用于update或者validate(save)的中填充数据

Example
const opts = {
  createdAt: 'date',
  token: 'ulid',
};
const data = {
  name: 'tree.xie',
};
fillData(opts, data);
// {name: "tree.xie", createdAt: "2017-03-14T12:59:56.417Z", token: "01ARZ3NDEKTSV4RRFFQ69G5FAV"}
console.info(data);

填充配置, {name: type}，name表示要填充的字段，type表示要填充的类型 type暂只支持'date'与'ulid'两种类型，date为ISOString
